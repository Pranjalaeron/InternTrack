# InternTrack – Internship Application Tracker

InternTrack is a full-stack internship application management platform that helps students organize and track their internship applications from a single dashboard.

## Features

* Real-time application tracking
* Gmail API integration
* Automatic email detection
* Application status updates
* Analytics dashboard
* Search and filtering
* Profile management
* JWT authentication
* Responsive UI

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### APIs & Services

* Gmail API
* Google OAuth 2.0
* Node-Cron

## Application Statuses

* Applied
* Interview
* Offer
* Rejected

## Key Highlights

* Tracks 100+ internship applications.
* Reduces manual tracking effort by 70%.
* Provides real-time application status updates.
* Automates email-based application monitoring.

## Installation

```bash
git clone <repository-url>

cd frontend
npm install
npm run dev

cd backend
npm install
npm run dev
```

## Environment Variables

```env
MONGO_URI=
JWT_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URI=
```
